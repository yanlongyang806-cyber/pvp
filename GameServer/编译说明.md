# GameServer 编译说明 - 全地图PVP版本

## 项目位置
```
I:\Night\Night\src\Night\GameServer\NNOGameServer.sln
```

## 在 Visual Studio 中打开

1. 打开 Visual Studio
2. 文件 → 打开 → 项目/解决方案
3. 选择：`I:\Night\Night\src\Night\GameServer\NNOGameServer.sln`
4. 等待项目加载完成
5. 选择配置：`Debug` 和 `Win32`
6. 右键解决方案 → 生成解决方案

## 依赖要求

项目需要以下依赖（位于 `I:\Night\Night\src\Night\`）：
- `CrossRoads\` - 包含 GameServerLib 和 Combat 相关代码
- `libs\` - 各种库文件（AILib, ContentLib, HttpLib, ServerLib, UtilitiesLib, WorldLib 等）
- `core\` - 核心库

## 全地图PVP功能

项目已包含 Combat 相关代码：
- `../../Crossroads/Common/Combat` - PVP 战斗系统
- `../../Crossroads/GameServerLib/Combat` - GameServer 战斗库

## 注意事项

- 如果依赖缺失，需要先准备这些目录
- 预构建事件可能需要 structparser.exe（如果缺失可以跳过）
- 编译成功后，可执行文件通常在 `Win32\Debug\GameServer.exe`
